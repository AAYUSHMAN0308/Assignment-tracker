# 📚 Assignment Tracker App

A simple and powerful Assignment Tracker web app built using HTML, CSS, and JavaScript.

---

## 🚀 Features

* ➕ Add assignments with deadline
* ✔ Mark tasks as completed
* 🗑 Delete assignments
* ✏ Edit assignments
* 🔍 Search assignments
* 📂 Filter (All / Completed / Pending)
* 📊 Progress tracking with chart
* 🌙 Dark mode support
* 📱 Mobile-friendly UI
* ⚡ Works offline (PWA support)

---

## 🌐 Live Demo

👉 https://aayushman0308.github.io/Assignment-tracker/

---

## 🛠 Tech Stack

* HTML
* CSS
* JavaScript
* Chart.js

---

## 📦 How to Use

1. Enter assignment title
2. Select due date
3. Click "Add"
4. Track your progress

---

This app can be installed as an Android app using PWA Builder.
